# cmd_interceptor
Command interception in go.

## Dependencies
 - Go compiler

## Build
```
chomd +x build.sh
$ ./build.sh
```

## Deployment

1. Move your command you want to intercept in the same folder and add the postfix '2'
2. Copy and rename the cmd_interceptor (for your architecture/os) to the same folder with the
   original cmd name
3. Enjoy
